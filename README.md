# My Baking Recipe Mock
[My Baking Recipe Mock](https://davidyang2149.github.io/MyBakingRecipe)은 Typescript를 포함하지 않은 버전입니다.

## Install dependencies

```sh
npm install
```

## Run dev server

```sh
npm run dev
```

## Run deploy in github

```sh
npm run deploy
```

## Lint and fix

```sh
npm run lint
```

## Run Coverage

```sh
npm run coverage
```

## Run e2e tests

```sh
# (Require: local server online) npm run dev 
npm run test:e2e
```
